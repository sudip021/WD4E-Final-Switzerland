# 🇨🇭 Explore Switzerland – Final Project

This is a responsive website created as part of the **Introduction to CSS3 Final Project**.  
The site showcases the beauty of Switzerland using modern HTML and CSS techniques.

---

## 🌍 Pages Included

- **Home (index.html)**  
  Introduction to Switzerland.

- **Gallery (gallery.html)**  
  Displays multiple images of Switzerland using a grid layout.

- **About (about.html)**  
  Provides general information about Switzerland.

---

## 🎨 Features Implemented

- ✅ **Skip to Main Content link** (visible on focus for accessibility)
- ✅ **Flexbox navigation bar**
- ✅ **CSS Grid layout** for image gallery
- ✅ **Box model styling** (margin, padding, border)
- ✅ **Hover effects** on navigation and images
- ✅ **nth-child selector** for styling alternate images
- ✅ **Responsive and clean layout**
- ✅ **Accessible design (WAVE tested)**
